1.5 Compiled Language vs. Interpreted Language

C++ is a compiled language
     code is processed to produce a low-level machine language executable
     must recompile and relink files if you edit any of them
if you move code to a different computer, you will usually need to recompile

many newer languages (python, java) are interpreted languages
     make changes and immediately run all of some of your code without waiting compilation
     designed for incremental development
     almost always runs slower than compiled programs

compilation is almost instantaneous for simple programs, so incremental editing is possible

C++ has more required punctuation than Python, and syntax is more restrictive
     compiler will read code and complain if there are syntax errors

1.14 Value Parameters and Reference Parameters

void month_and_day(int julian_day, int year, int &month, int &day)
     first two are value parameters
     do not change the corresponding argument in the calling function
     copies are created
     
     reference parameters are aliases
     changes the corresponding variables in the calling functions

when a function is used to provide just one result
     make that result the return value of the function and pass other parameters by value
when a function needs to provide more than one result
     these results should be returned using multiple reference parameters

1.16 About STL String Objects

string is an object type defined in the library to contain a sequence of characters

multiple ways to construct:

if you create another string from a string, it doesn't create an alias
     changes to one doesn't affect the other

     returns the size of the string
     length can do the same thing

input to string objects through streams (console or files):

"+" is defined on strings (concatenation)
"=" overwrites the current contents of the string

individual characters can be accessed using subscript operator much like an array

strings define a special type string::size_type
     the type returned by the string function size()
     generally equivalent to unsigned int
